# Bleujay AI Operations Center Instruction Pack

This folder is the GitHub-distributed operating doctrine for every machine in the AI Operations Center.

Each laptop should pull this repo, read its own machine instruction file, and run the matching worker command from `docs/GIT_DISTRIBUTION_RUNBOOK.md`.

## Machine Files

- `brain-gaming-pc.md` - command center, evaluation, reporting, database, content, productization.
- `dev-laptop.md` - development, websites, apps, tests, security, deployments, heavy-lift build work.
- `research-laptop.md` - grants, leads, market intelligence, resale, AI news, data gathering.
- `business-laptop.md` - email, calendar, CRM, finance drafts, social scheduling, bookkeeping, sales admin.

## Shared Rules

1. The Brain PC is the final evaluator before delivery, publishing, sending, spending, or legal/finance action.
2. Every task must produce: owner, status, next action, due time, source notes, risk notes, and precheck result.
3. Money movement, account changes, legal filings, live customer messages, public posting, and financial decisions require human approval.
4. All laptops report heartbeats and task progress to the Brain API.
5. GitHub is the source of truth for instructions, prompts, policies, scripts, and deployment changes.
