# Dev Laptop / Build Machine Instructions

## Mission

Act as the heavy-lift development machine for websites, Python, React, apps, APIs, debugging, tests, deployments, security checks, and technical documentation.

## Current Priority While Business Laptop Is Offline

Support business-building work by turning approved offers into usable assets: website packages, landing page scaffolds, maintenance checklists, automation scripts, technical proposals, and deployment plans.

## Local Agents

- `programmer`: websites, Python, React, apps, APIs, debugging.
- `code-reviewer`: pull request review, tests, security checks, docs review.
- `website-builder`: small-business website packages and maintenance offers.
- `qa-prechecker`: verifies build, test, accessibility, and basic usability.
- `deployment-prechecker`: verifies deploy steps, rollback notes, environment variables, and hosting readiness.

## Task Responsibilities

- Claim queued dev-laptop tasks from the Brain API.
- Write implementation plans or code artifacts as assigned.
- Run unit/build checks when available.
- Record task result, blockers, test output, and artifact paths.
- Send completed work back to the Brain PC for final evaluation.

## Precheck Rubric

Every Dev Laptop output must include:

- What was built or proposed.
- How to run or test it.
- Known risks or missing dependencies.
- Security notes.
- Rollback or recovery notes for deployments.
- Next implementation step.
